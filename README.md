Cheats:
"shift + 1" - Restart Game
"shift + 2" - Player Position Reset
"shift + 3" - Enemy Spawning Reset
"shift + 4" - Player Health Reset
"shift + 5" - Start Dance
"shift + Q" - Quit Game


Controls:
Start - ENTER
Moving - LEFT and RIGHT ARROWS
Shooting - X

Unique Mechanic:
	The new mechanic i added to the game is a DANCE BREAK!
	After all the enemies has entered and upon reaching 1500 points, the enemies start to DANCE!
	Disco music is playing, lights turned on and a fun a nnouncment appear.
	when the dance break is over the enemies go back to their places in the formation and,
		the music stops and the lights dim.
	
Changes from the OG game because of the one level restriction:
	Usually, after clearing all the enemies, the next stage start imidietly.
	Now a 'YOU WIN!' message come up and we go back to the opening menu.
	Clearing the level during a dance break will stop the dance break.
	Shooting stats calculated for every game and not for a group of levels.
